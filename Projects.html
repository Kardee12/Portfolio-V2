The Marble Sorter

 We will start by inserting 15 marbles (of varying type, size, color, density, and material) into our “marble sorter.” Our device will then proceed to sort the 15 marbles based on three characteristics: density, magnetism, and size. In this project, I used my programming and problem solving skills to sort the wooden marbles while using a net. I learned how to program an ultrasonic sensor to sense if there are any marbles left.  Find out more by pressing on the link below!

The Heart Valve

The goal of this project is to design, print, and test a heart valve with information given from PBS for Anna Garcia so that it may be used later for other patients. The heart valve has to be able to allow liquids to flow in one direction but nothing may pass through the other way. I learned how to properly scale and model an object while utilizing Autodesk Inventor 2017. I used my problem-solving skills to sketch the heart-valve.
